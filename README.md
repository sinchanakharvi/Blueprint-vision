# AI-Based Blueprint Analysis for Intelligent House Color Visualization

## Overview

AI-Based Blueprint Analysis for Intelligent House Color Visualization is an intelligent web application that transforms traditional architectural blueprints into interactive 2D and 3D visualizations. The system leverages Computer Vision, Deep Learning, and Artificial Intelligence to automatically detect rooms and structural elements, generate intelligent color recommendations, and estimate paint requirements and project costs. It provides architects, interior designers, and homeowners with an efficient way to visualize and evaluate design decisions before construction or renovation.

---

## Features

- Upload architectural blueprint images
- Automatic blueprint preprocessing
- AI-based room and structural element detection
- Intelligent color palette generation
- Room-wise color customization
- Interactive 3D building visualization
- Paint quantity estimation
- Paint cost estimation across multiple brands
- Download visualization and budget report
- Responsive and user-friendly interface

## System Workflow

1. Upload the blueprint image.
2. Preprocess the blueprint for analysis.
3. Detect rooms, walls, doors, and windows using AI.
4. Generate intelligent color recommendations.
5. Apply selected colors to the blueprint.
6. Generate an interactive 3D visualization.
7. Estimate paint quantity and project cost.
8. Download the final visualization and budget report.

## Installation

### Clone the repository

```bash
git clone https://github.com/yourusername/AI-Blueprint-Visualization.git
```

### Navigate to the project directory

```bash
cd AI-Blueprint-Visualization
```

### Install backend dependencies

```bash
pip install -r requirements.txt
```

### Run the backend server

```bash
uvicorn main:app --reload
```

### Install frontend dependencies

```bash
npm install
```

### Run the frontend

```bash
npm start
```
